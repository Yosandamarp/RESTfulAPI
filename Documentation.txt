menggunakan microsoft word,google doc,samapai menggunakan tool khusus untuk
dokumentasi API. Sangat disarankan untuk menggunakan tool khusus dokumentasi
API.

Swagger

Open API adalah spesifikasi yang banyak diadopsi oleh industri untuk 
mendeskripsikan API
Keuntungan menggunakan OpenAPI adalah 
Standard,sehingga bisa dibuat oleh programmer bahasa pemrograman apapun

melakukan generate kode client dari OpenAPI Specificastion.
